# Configuração do plugin Gradle do ANTLR4  em um projeto Kotlin

O Gradle possui um plugin que facilita o processo de geração de código do ANTLR4.

Primeiramente deve-se instalar o plugin ANTLR do IntelliJ IDEA para ter suporte aos arquivos de gramática (extensão **g4**).

Depois realizar a seguinte configuração no arquivo **build.gradle.kts**

```kotlin
plugins {
	// ...
	antlr  // Declaração do plugina ANTLR do Gradle
}

// ...

dependencies {  
	antlr("org.antlr:antlr4:4.13.0")  
	// ...
}

// ...

tasks.generateGrammarSource {  
	// Tamanho máximo do heap
	maxHeapSize = "64m"

	// Diretório de saída após geração do código 
	outputDirectory = File("src/main/<pacote-aplicação>/antlr")

	// Argumentos de compilação do ANTLR
	// arguments = arguments + listOf("-visitor", "-long-messages")  
}
```

Criar o diretório `src/main/antlr` para armazenar os arquivos de gramática (extensão **g4**).

Exemplo de arquivo de gramática 

```text
grammar Hello; // Define uma gramática chamada Hello  
  
@header {  
package <pacote-aplicação>;  
}  
  
r : 'hello' ID; // Realiza correspondência entre a palavra-chave hello e um identificador  
ID : [a-z]+; // Realiza a correspondência de identificadores com letras minusculas  
WS : [ \t\r\n]+ -> skip; // Ignora os caracteres de espaço, tabulação, nova linha e retorno de carro
```


Para gerar os arquivos com o ANTLR a partir do arquivo de gramática, executar a tarefa **generateGrammarSource** do Gradle.
