# Instalação do ANTLR4 no Arch Linux

Para instalar o ANTLR4 no Arch Linux basta executar o seguinte comando:

```bash
paru -S antlr4
```

Após finalizar o processo de instalação, estarão disponíveis os comandos `antlr4` e `grun`
