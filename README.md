# My Second Brain

Este repositório tem como objetivo armazenar as minhas notas seguindo a metodologia **Zettelkasten** adaptada as minhas necessidades de pesquisa. 

Utilizo o [Obsidian](https://obsidian.md/) como software de gestão e, consequentemente, os arquivos estão escritos em [markdown](https://www.markdownguide.org/).

A forma que este repositório está organizada compreende meu fluxo de trabalho e a minha interpretação do **Zettelkasten**, logo pode variar de outros fluxos de trabalhos de outras pessoas. O importante é que funcione para mim (por isso chamo de segundo cérebro).
